# CloudScan Oral AI

CloudScan Oral AI is an academic healthcare-focused web-based oral cancer detection platform designed to assist in early diagnosis through Convolutional Neural Network (CNN)-based medical image classification.

## Project Overview
The platform enables users to securely upload oral cavity medical images, preprocess them, and classify them into Normal or Abnormal categories using deep learning. The project integrates backend APIs, machine learning pipelines, dataset structuring, and frontend dashboard planning.

## Key Features
- Secure medical image upload
- CNN-based oral cancer prediction
- Image preprocessing pipeline
- Model training and evaluation scripts
- Flask backend architecture
- Frontend healthcare dashboard structure
- Modular project design
