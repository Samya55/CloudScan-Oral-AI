# Frontend Module

This directory contains the planned healthcare dashboard frontend for CloudScan Oral AI.

## Planned Features
- Secure oral image upload interface
- Prediction result display
- Confidence score dashboard
- Responsive medical UI
- React-based healthcare dashboard
- API integration with Flask backend

## Planned Tech Stack
- React.js
- HTML/CSS/JavaScript
- Tailwind CSS
- Axios
- Healthcare dashboard UI

## Future Scope
- Grad-CAM visualization
- Patient history
- Multi-class oral disease reporting
- Cloud deployment integration
